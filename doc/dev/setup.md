# Setup

When you execute
```
$ source setup.sh
```
The following environment variables are defined:

1. `$DEV`: Root
2. `$B`: Builder
3. `$S`: Source
3. `$SL`: Library Source
3. `$SC`: Client Source
3. `$SS`: Server Source
4. `$T`: Test

and we have the aliases:

1. `builder`: Build script
2. `client`: Client
3. `server`: Server
